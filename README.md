# TrendsAppWebview
## Android Trends &amp; Hoax Checker App Webview with Dark Mode Support

### Features
- Shows 4 Tab 
  - Google Trends
  - Youtube Trends 
  - Twitter Trends 
  - TurnBackHoax.id
- Pull to reload page
- Movable Share button
- Dark Mode Support
- Easy to change Trends Region
- Special Effect on About Dev's Page (only appears on Dev's Birthday)
